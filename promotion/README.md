# Promotion Materials

This directory stores investor and partner communication drafts for OpenChipAgentPlatform.

Current files:

- `investor_partner_deck_draft.md`: slide-by-slide narrative draft.
- `talent_profiles.md`: early team hiring profile for Agent and large-model talent.
- `sources.md`: public sources and evidence used by the draft.

The material is written as Markdown first so it can be reviewed quickly and later converted into PPT.
