# Brainstorm

This directory records exploratory product ideas for OpenChipAgentPlatform.

Unlike `requirements/`, which preserves the original user requirements, and `research/`, which records structured investigation, this directory is for open-ended product brainstorming.

## Documents

- [01-harness-hermes-loop-goal.md](01-harness-hermes-loop-goal.md): brainstorming notes on combining chip design agents with harness engineering, Hermes-style learning, loop engineering, and goal-driven autonomous workflows.
- [02-open-agent-base-for-chip-design.md](02-open-agent-base-for-chip-design.md): brainstorming notes on building a chip-design-specific agent platform on top of open-source coding agents such as OpenCode and Hermes.
