# Goal

Use the OpenChipAgent Harness/Loop idea to drive `TargetIt/openPwmChipFlow` through the full chip flow without skipping Phase 3 through Phase 6.

Required stages:

- project probe;
- RTL simulation;
- synthesis;
- place and route;
- DRC/LVS/Antenna physical verification;
- GDS delivery;
- evidence review;
- Harness/Loop lesson capture.

Completion requires real tool evidence and generated backend artifacts. A stage cannot be marked complete only because a report exists.
